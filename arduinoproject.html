My Arduino Project

This was an Arduino project that I created. It is a simple circuit that displays a number on the seven segment display when I push the corresponding number on the keypad. There are quite a bit of wires, because the seven segment display is hooked up to the shift register. The shift register receives bits of data from the microboard, and it uses these bits of data to turn on certain pins. This turns on the seven segment display and shows the number we want.
